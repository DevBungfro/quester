Just like [Type](/api/main/type/), these are used to specify what type of time you want the `Quester.Type.Timer` Quest to countdown in.

Below is a list of all built-in timers for Quester

---

Name   | Type | Value
 ---    | ---
 Minutes | string | 0
 Seconds | string | 1
 Custom | string| 2
